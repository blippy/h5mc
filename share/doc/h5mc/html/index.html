sharelock holmes stats list

Imports /home/mcarter/cubie/home/mcarter/data/StatsList.csv into sharelock.h5 in the same directory, giving it a datestamp corresponding to the creation date of the CSV file. the file should NOT be preporcessed first. It is able to work out if the columns are strings or numerica data (string data always begins with a double-quote).

Interesting files for you to look at when considering further projects:

  • shsl.c - contains complete HDF5 processing, and is able to process string as well as doubles
  • shsl.l and shsl.y are lex and yacc file that read the CSV data
